# json-streaming

You can stream your json file to send request to any service, and get data in CSV format. 


## Run Locally

Clone the project

```bash
  git clone https://github.com/yeshalilsahin/json-streaming.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
